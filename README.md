# Frame x Frame — Houses and Stories

Cinematic films about India's most extraordinary homes. Himalayan retreats, heritage houses, mountain villas — watched by 500K+ viewers worldwide.

🔗 **Live site:** [framexframe.online](https://framexframe.online)  
📺 **YouTube:** [@weareframexframe](https://www.youtube.com/@weareframexframe)

---

## Project Structure

```
project/
├── index.html          # Main HTML file (all pages)
├── css/
│   └── styles.css      # All styles
├── js/
│   └── script.js       # All JavaScript (film data, navigation, FAQ)
└── assets/
    ├── images/
    ├── icons/
    └── fonts/
```

## How to Run

No build step needed. Just open `index.html` in a browser, or serve with any static file server:

```bash
# Python
python3 -m http.server 8080

# Node (npx)
npx serve .
```

## Tech Stack

- **HTML5** — semantic markup, SEO meta tags
- **Vanilla CSS** — custom properties, responsive grid, animations
- **Vanilla JS** — SPA navigation, YouTube embed, IntersectionObserver reveal
- **Fonts** — Cormorant Garamond + DM Sans via Google Fonts
